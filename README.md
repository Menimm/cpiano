# פסנתרון HERO - גרסת ילדים v4

## מה חדש
- חלון **הגדרות נפרד** לאימון (BPM, חלון פגיעה, קו רץ בחוברת תווים).
- תווים נופלים עם שם תו באות גדולה.
- כשהתו מגיע לראש הקליד: הוא הופך לירוק ואז נעלם.
- קליד נצבע ירוק כשזוהתה פגיעה נכונה (כולל מיקרופון), ואדום כשיש פספוס/זיהוי שגוי.
- פסנתר 61 קלידים בתחתית עם מיפוי מדויק של מיקום התו.
- תצוגת חוברת תווים עם קו רץ שמראה מתי לנגן.
- נוספה מנגינת **הפנתר הוורוד** לרשימת השירים.

## שירים
- ניצנץ כוכב קטן
- Mary Had a Little Lamb
- Ode to Joy
- הפנתר הוורוד (קטע פתיחה)
