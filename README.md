# פסנתרון HERO 🎹

אפליקציית Web בעברית ללימוד פסנתר/אורגנית בחוויה אינטראקטיבית בסגנון Guitar Hero.

## מה השתפר משמעותית
- אנימציה רציפה על המסך (תווים נופלים כל הזמן).
- חוויית "וידאו-משחק": נתיבים, אזור פגיעה, קומבו, שלבים ופידבק חי.
- תצוגת חוברת תווים ברורה (SVG) עם תו נראה תמיד + הסבר מילולי.
- מצבי קלט:
  - **מקלדת/לחיצה**: מקשים 1-8 או לחיצה על התווים במסך.
  - **מיקרופון**: נגינה אמיתית עם זיהוי תו בזמן אמת.

## הרצה מקומית
```bash
python3 -m http.server 4173
```
ואז לפתוח:

http://localhost:4173

> כדי להשתמש במיקרופון, יש לאשר הרשאה בדפדפן.

## הערות פריסה (GitHub Pages / Cache)
- הממשק מציג תג גרסה בראש העמוד (`Build: hero-2026-03-12`).
- אם אתם לא רואים את תג הגרסה הזה - אתם כנראה על קוד ישן או Cache ישן.
- בצעו Hard Refresh (`Ctrl+Shift+R`) או פתחו בחלון אינקוגניטו.
