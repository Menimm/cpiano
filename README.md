# פסנתרון 🎹

אפליקציית Web בעברית ללימוד פסנתר/אורגנית מאפס, כולל האזנה בזמן אמת מהמיקרופון לזיהוי תווים.

## מה יש באפליקציה
- מסלול לימוד מדורג משלב היכרות עם קלידים ועד יצירות בסיסיות.
- ממשק פשוט בעברית (RTL) שמתאים לילדים בגיל 9.
- משוב מיידי בזמן אמת על התו שמנוגן.
- אזור "לומדים לפני שמתרגלים":
  - הסבר מה אומר כל תו (למשל C4 = דו אמצעי).
  - חמשה אינטראקטיבית המציגה את התו כמו בחוברת תווים.
  - מיני-מקלדת שניתנת ללחיצה כדי לבחור תו ללמידה/תרגול.
- אלמנטים מעודדי מוטיבציה: כוכבים, רצף יומי והתקדמות בשלבים.

## עקרונות UX שיושמו (בהשראת אפליקציות לימוד מוזיקה מצליחות)
1. קודם מושג + הדגמה חזותית, ואז תרגול.
2. שיעורים קצרים וברורים (5-8 דקות).
3. משוב מיידי אחרי כל ניסיון.
4. גיימיפיקציה עדינה ולא מלחיצה.
5. שפה פשוטה ועיצוב ידידותי לילדים.

## הרצה מקומית
מכיוון שמדובר באפליקציית HTML/CSS/JS סטטית, אפשר להריץ כל שרת מקומי:

```bash
python3 -m http.server 4173
```

ואז לפתוח:

http://localhost:4173

> שימו לב: הפעלת מיקרופון דורשת פתיחה דרך localhost/https והרשאה מהדפדפן.
